Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://wilson-magic.github.io/repo/ (Give it the name "Magicality").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Magicality", and install repository.magicality-0.0.4.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Magicality Repo"
  • Go into the "Video add-ons" section in the repo, and you'll find Masterani Redux

Enjoy!
